Mis Proyectos

A continuación encontrarás una selección de proyectos académicos y personales que he desarrollado durante mi formación en la Universidad CESUN. Cada proyecto representa un desafío único y una valiosa oportunidad de aprendizaje en mi camino hacia convertirme en un desarrollador de software competente.

🤖 Agente de IA con N8N

Sistema de automatización inteligente que combina N8N para flujos de trabajo con capacidades de IA para procesamiento y toma de decisiones automatizadas.

Tecnologías: N8N, Python, APIs de IA, Docker, Node.js

Estado: En desarrollo

🚀 Gestión de Facturación - API Backend

API backend completa para gestión de facturación empresarial con autenticación de usuarios, gestión de proveedores y sistema de facturas integrado.

Tecnologías: FastAPI, Python, SQL Server, Uvicorn, PyODBC

Estado: En desarrollo

🎓 Agente Informativo CESUN

Chatbot inteligente que proporciona información académica, de servicio comunitario y calendario escolar de la Universidad CESUN de forma interactiva.

Tecnologías: Python, APIs de IA, Base de datos, Web Scraping

Estado: En desarrollo

👥 CRUD Gestión de Clientes

Aplicación de escritorio para gestión completa de clientes con operaciones CRUD, desarrollada en Visual Studio 2022 con arquitectura moderna.

Tecnologías: C#, Entity Framework, Windows Forms, SQL Server

Estado: Completado

✅ To-Do List con FastAPI

Aplicación de gestión de tareas con API REST moderna, base de datos relacional y contenedorización Docker para despliegue simplificado.

Tecnologías: FastAPI, Python, PostgreSQL, Docker, SQLAlchemy

Estado: Completado
